# 环境搭建  
npm install  
npm run dev

# build for production with minification
npm run build  
add <script src="./dist/build.js"></script>  to index.html  
open file in browser  

# 功能
打开app.vue  
* allTree.vue: 全量目录树
* lazyTree.vue: 懒加载树
* test1.vue: el-select远程搜索，回车时选择框消失处理
* selectWidth.vue: el-select下拉框大于输入框
